# Pinpin

A calming Chrome extension for new tabs.

```bash
git clone git@github.com:dentednerd/pinpin.git
cd pinpin
yarn
yarn start
```

## To test in Chrome

Run `yarn build`.

In Chome, open `chrome://extensions/`.

Switch on developer mode.

Click "Load unpacked".

Navigate to `/build`.

Open a new tab.
